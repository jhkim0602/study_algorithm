# 자료구조·알고리즘 학습노트 (NotebookLM 영상 제작용)

이 폴더 하나만 있으면 NotebookLM으로 학습 영상을 만들 수 있습니다.

## 폴더 구성
| 파일 | 용도 |
|---|---|
| `00_전체_NotebookLM학습노트.md` | 4단원 전체 (한 영상으로 만들 때) |
| `CH06_정렬알고리즘.md` | 단원별 (정렬) — 단원당 영상 추천 |
| `CH07_문자열검색.md` | 단원별 (문자열 검색) |
| `CH08_리스트.md` | 단원별 (리스트) |
| `CH09_트리.md` | 단원별 (트리) |

각 파일에는 **개념 정리 → 개념별 시각화 → 160문제의 풀이·정답 근거**가 들어 있습니다.
코드 문제는 실행 추적표, 트리 문제는 그림(mermaid), 복잡도는 표로 시각화돼 있습니다.

## NotebookLM에서 영상 만드는 법
1. NotebookLM에 새 노트북을 만든다.
2. 소스 업로드:
   - 전체 한 영상 → `00_전체_NotebookLM학습노트.md` 1개
   - 단원별 영상(추천) → `CH06~CH09` 파일을 각각 (문제를 빠짐없이 다룰 확률이 높음)
3. 언어를 **한국어**로 설정한다.
4. Studio 패널 → **동영상 개요(Video Overview)** → **맞춤설정(Customize)** 칸에 아래 프롬프트를 붙여넣고 생성한다.

## 붙여넣을 프롬프트
```
이 자료로 '시험대비 강의' 동영상을 만들어줘. 대상은 자료구조·알고리즘 시험을 준비하는 학생이야.

진행 방식:
1) 먼저 이 단원의 핵심 개념을 비유를 들어 1~2분 안에 빠르게 정리해줘.
2) 그다음 문제(Q번호)를 처음부터 끝까지 하나씩, 빠짐없이 다뤄줘. 건너뛰지 마.
3) 각 문제마다: (a) 문제를 짧게 읽고 (b) 정답이 무엇인지 (c) "왜 이게 답인지"를 가장 쉬운 말로 설명하고 (d) 나머지 보기가 왜 틀렸는지 한 줄로 짚어줘.
4) 코드 문제는 자료에 있는 '실행 추적표(배열이 단계별로 바뀌는 표)'를, 트리 문제는 그림을 화면에 보여주며 천천히 따라가줘.

말투는 친근하고 또박또박, 너무 빠르지 않게. 전문용어는 쉬운 비유로 풀어줘. 한국어로.
```

## 팁
- **오디오(팟캐스트)** 도 같은 프롬프트로 "오디오 개요"를 만들면 이동 중 듣기용으로 좋다.
- 특정 알고리즘만 더 깊게 → 프롬프트 끝에 `특히 퀵 정렬·병합 정렬을 더 자세히`처럼 덧붙인다.
- 영상이 일부 문제를 건너뛰면 → 그 단원 파일을 객관식/코드 파트로 더 쪼개서 만들면 누락이 준다.
