# 자료구조 · 알고리즘 학습 위키

PDF 문제를 전수 수록한 정적 학습 위키입니다. **개념 정리 → 인터랙티브 시각화 → 문제 풀이**를 한 화면에서 학습할 수 있습니다. 전공자(CS) 대상, 한국어.

## 수록 범위 (총 160문제)

| 챕터 | 문제 수 | 구성 |
|---|---|---|
| Ch06 정렬 알고리즘 | 40 | 객관식 20 + 코드형(Python) 20 |
| Ch07 문자열 검색 | 40 | 객관식 20 + O/X 20 |
| Ch08 리스트 | 40 | 객관식 20 + O/X 20 |
| Ch09 트리 | 40 | 객관식 20 + O/X 20 |

> 모든 문제·정답·해설은 원본 PDF에서 그대로 옮겼으며, 정답은 원본 정답표와 1:1 교차 검증을 통과했습니다.
> 정렬 PDF 2세트는 Q4만 다른 동일 문제라 `Q4수정` 정본 1세트로 통합했습니다.

## 주요 기능

- **위키 디자인** — 좌측 사이드바 네비게이션 + 가독성 중심 본문, 라이트/다크 테마
- **세부 학습 페이지 21개** — 각 정렬(버블·선택·삽입·쉘·퀵·병합·힙·도수)·검색(브루트포스·KMP·보이어무어)·리스트(배열·단순·이중·원형·커서)·트리(용어·순회·이진트리·BST·BST삭제)를 **주제별 전용 페이지**로. 개념→참조 코드→복잡도 표→전용 시각화→관련 문제를 한 페이지에서 학습. (코드 문제도 해당 알고리즘 페이지에 연결)
- **풀이 모드** — 보기/OX를 클릭하면 즉시 채점(정답·오답 표시 + 해설 자동 노출, 다시 풀기). 점수·오답노트 없이 학습 중심.
- **세분화 개념정리** — 챕터마다 정의→원리→예시→복잡도/주의로 잘게 나눈 심화 본문(개념 블록 255개) + 비교표
- **상세 해설** — 160문제 전부에 *자세한 풀이* 수록: 핵심 요약, 단계별 풀이(코드 문제는 한 줄씩 실행 추적), 보기별 정답/오답 분석, 핵심 개념, 함정. (정답·코드 추적은 독립 에이전트로 적대 검증)
- **통합 문제(Practice)** — 160문제를 **대단원·개념·유형**으로 자유 조합하고 **순서 섞기**로 풀이. 모든 조합 가능.
- **인터랙티브 시각화 13종 (SVG · 직접 조작 · 단계 재생)**
  - 정렬: 8종 단계 실행기 · **복잡도 비교(n별 막대)** · **안정성 시각화**
  - 문자열 검색: 브루트포스/KMP 단계 실행기 · **보이어-무어 단계** · **BF vs KMP 비교 횟수**
  - 리스트: 단순·이중·원형 조작 실습 · **배열 vs 연결리스트 비용** · **커서/free list**
  - 트리: BST 삽입·삭제·검색·순회 · **완전이진트리↔배열 인덱스** · **순회 비교** · **삽입 순서와 균형/편향**
- **문제 풀이** — 정답·해설 펼치기 토글, 풀이에 필요한 **개념 태그**, 챕터 단위 전체 펼치기/접기
- **코드 가독성** — Python 코드 문제는 구문 강조 + 복사 버튼
- **DB 없음** — 테마·정답 펼침 상태만 `localStorage`에 저장

## 실행

```bash
npm install
npm run dev      # 개발 서버 (http://localhost:5173)
npm run build    # 정적 빌드 → dist/
npm run preview  # 빌드 결과 미리보기
```

`dist/`는 순수 정적 파일이라 어떤 정적 호스팅(또는 `file://`)에서도 동작합니다.

## 구조

```
src/
├─ data/
│  ├─ chXX-*.js     # 챕터별 { 문제[] } (원본·검증 완료, 손대지 않음)
│  ├─ concepts/     # 챕터별 세분화 개념 (concept 객체)
│  ├─ details/      # 챕터별 상세 해설 (문제 no → detail)
│  ├─ subtopics/    # 세부 학습 페이지 (주제별 blocks + conceptTags)
│  └─ index.js      # 합성: 개념·해설·세부주제 결합, 통합 문제용 평탄화/개념목록
├─ components/       # Sidebar, ChapterView, ProblemCard, ProblemDetail, ConceptSection, CodeBlock, MiniTree, Home, Practice
│  └─ viz/           # 시각화 13종 + StepPlayer(공용 단계 재생기)
├─ hooks/            # useLocalStorage, useHashRoute
└─ styles/           # global.css (테마 변수 · 위키 스타일 · hljs 토큰)
```

- 문제 추가: `src/data/chXX-*.js`의 `problems` 배열에 객체 추가.
- 해설 보강: `src/data/details/chXX-*.js`의 해당 번호 `detail` 수정.
- 개념 보강: `src/data/concepts/chXX.js`의 `blocks` 수정.
